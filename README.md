# Random Quote Generator

A simple web project that generates a random inspirational quote each time the user clicks a button. This project was built for practicing core **JavaScript, HTML, and CSS concepts**.

## 🚀 Features
- Generates a random quote on button click
- Clean and responsive UI
- Built using pure HTML, CSS, and JavaScript
- Beginner-friendly project for practicing DOM manipulation

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
random-quote-generator
│
├── index.html
├── style.css
└── script.js


## 💡 How It Works
- A list of quotes is stored in JavaScript.
- When the user clicks the **Generate Quote** button, JavaScript selects a random quote from the list.
- The selected quote is then displayed on the webpage using DOM manipulation.

## ▶️ How to Run the Project
1. Clone the repository
2. Open the project folder
3. Run the `index.html` file in your browser
